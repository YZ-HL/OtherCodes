### 简单介绍

一个帮助你在电脑上查成绩的脚本。

在学校不方便用手机时可能会用上。

### 注意事项

**输入密码时不会显示密码，如果拿回班内使用记得强调这点。**

最近好像有库的更新，函数名换了，如果发现原代码运行不了请尝试最近更新带``new``的代码。
